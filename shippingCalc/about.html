About

This application is used to calculate the shipping costs for particular zip code and weight combinations. It queries two tables within a database to first analyze which zone the selected zip code corresponds to, and second to provide with the correct cost based on a zone and weight combination. The top portion of the home page is stylistically close to what I desired, however is not functional. The bottom portion of the home page is functional, yet stylistically cringe-worthy. Five separate forms must be submitted (in order) so that the database and given tables can be created, and populated using individual PHP files. Next, a ZIP code and a package weight can be selected (from a selection based on the appropriate data) and the cost can be returned after the submit button is pressed. Unfortunately the result is displayed on a new page, rather than displayed on the same page without a page load.
