About

This application is used to calculate the shipping costs for particular zip code and weight combinations. It queries two tables within a database to first analyze which zone the selected zip code corresponds to, and second to provide with the correct cost based on a zone and weight combination. The database for which the program queries and returns the data should already be initialized and populated by the time the user is able to make selections.

The first window on the home page instructs the user to create the database by clicking a button, however this is stylistically implemented rather than functionally implemented. Next, a ZIP code and a package weight can be selected (from a selection based on the appropriate data) and the cost can be returned after the submit button is pressed. The result is displayed on the same page, providing the user with the ability to make another query.
